# SocialMedia Project
Create a social media similar to **Instagram** with **ReactJS**, **NodeJS**, **MongoDB**
and **Bootstrap**.

### MongoDB database
* Name `socialmedia`
**Collections**
* Comments
* Followers
* Likes
* Publications
* Users

### Running
* Clone repository
* Run MongoDB Server
* Run `npm install`
* Run `npm start`
* Go to `http://localhost:3000`
